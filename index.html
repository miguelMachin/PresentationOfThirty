DEFINICIÓN DEL PROYECTO

¿Qué es thirty?
Es una pequeña red social.
¿Para qué sirve?
Permite la comunicación entre diferentes personas del país o diferentes países y es un lugar donde pueden intercambiar información, opiniones, vivencias, etc.

ALCANCE DEL PROYECTO

¿Para quién es?
Es una aplicación para todo tipo de usuarios.
¿Qué ofrece?
  • Cifrado de contraseña
  • Búsqueda, agregado y borrado de usuarios
  • Compartir estados
  • Publicar información, opiniones, etc
  • Añadir a favoritas las publicaciones
  • Sistema de notificaciones

TECNOLOGÍAS USADAS

  • Nginx

    Es un servidor web no visto en el ciclo y se usa como proxy en esta aplicación.

  • Nodejs

    Es un entorno en tiempo de ejecución multiplataforma y permite escribir todo el back-end en js.

TECNOLOGÍAS USADAS

  • Express

    Framework de lado servidor que facilita el desarrollo del back-end con node.

  • MongoDB

    Base de datos no relacional no vista en el curso.

TECNOLOGÍAS USADAS

  • Mongosse

    Libreria para conectar con mongodb y permite hacer consultas relacionales.

  • Handlebars

    Motor de plantillas

  • HTML-CSS

    Permite representar la información.

RECURSOS USADOS

  • Sistema: Ubuntu Gnome 18.04
  • Visual Studio Code
  • Github
  • Google chrome y Firefox
